FantasyRally — URLs & Views

Use these URLs on your host (replace your-host with your domain or IP):

  • App (web / mobile web): http://your-host/
  • Admin portal: http://your-host/admin
  • API base: http://your-host:4000 (or internal: http://api:4000 in Docker)

Frontend views (SPA pages)

The frontend exposes the following main views (select from the app navigation):

  • Viewer — main content gallery / list (default)
  • 3D Park — interactive 3D viewer / park scene
  • Garage — user garage / collection
  • Admin — administrative UI (requires admin access)
  • Account / Auth — sign in / sign up / account details
  • Payment / Deposit — create Stripe Checkout or PaymentIntent to deposit sweepcoins

Backend endpoints (selected)

  • /api/content — list uploaded content
  • /api/upload — POST file uploads (multipart/form-data)
  • /api/signup, /api/signin — auth
  • /api/me — current user (requires Bearer token)
  • /api/create-checkout-session — create Stripe Checkout session (requires auth)
  • /webhook — Stripe webhook endpoint (set STRIPE_WEBHOOK_SECRET if using signatures)

Notes

  • The admin UI is served at /admin and is a separate entry built as admin.html.
  • If you run the Docker compose setup, the web UI is on port 80 and the API on port 4000.
  • For mobile development use the Vite dev server and open http://<your-ip>:5173 on your phone.
